# GamerGate — Маркетплейс игровых ключей

## О проекте
**GamerGate** — это современная платформа для покупки цифровых ключей к видеоиграм. Наш проект позволяет пользователям быстро находить любимые тайтлы, пополнять кошелек и совершать безопасные покупки для активации в различных игровых клиентах (Steam, Rockstar Games, EA и др.).

## Основные возможности
* **Удобный поиск:** Быстрый поиск игр по каталогу.
* **Личный кабинет:** Управление балансом и просмотр списка желаемого.
* **Безопасная корзина:** Простой и понятный процесс оформления покупки.
* **Разнообразие методов оплаты:** Возможность оплаты с привязанной карты или внутреннего баланса кошелька.
* **Адаптивный дизайн:** Корректное отображение сайта на любых устройствах.

## Используемые технологии
Проект разработан с использованием классического стека веб-технологий:
* **HTML5:** Семантическая структура страниц.
* **CSS3:** Адаптивная верстка, использование Flexbox для гибкого позиционирования элементов.
* **JavaScript:** Логика работы корзины, динамическое обновление данных и взаимодействие с интерфейсом.
